### **Project Overview**

**Samvadini** is an **AI-powered assistive communication web application** developed under the **ASEP2 initiative at VIT Pune (2024–25)**. The platform is designed to empower persons with disabilities by reducing communication barriers through the integration of **Machine Learning (ML)**, **Natural Language Processing (NLP)**, and modern web technologies.

The application provides an accessible and inclusive interface that enables users to communicate more effectively across different languages and communication formats.

---

### **Key Features**

* 🤟 **Indian Sign Language (ISL) Conversion**
* 🔊 **Emotion-Aware Text-to-Speech**
* 🌐 **Multilingual Language Translation**
* 🤖 **AI-Powered Chat Assistant**
* 🌦️ **Real-Time Weather Information**
* 😂 **Joke Generator**
* 📚 **Wikipedia Search & Information Retrieval**

---

### **Project Versions**

Samvadini has evolved through **three polished versions**, each introducing improved accessibility features, enhanced UI/UX, and expanded AI capabilities.

## **Installation & Setup**

```bash
git clone https://github.com/Prasad-Dalvi/Samvadini.git

cd Samvadini

python -m venv venv

# Windows
venv\Scripts\activate

pip install sentencepiece-0.2.0-cp312-cp312-win_amd64.whl

pip install -r requirements.txt

python app.py
```

After starting the application, open your browser and navigate to:

```
http://127.0.0.1:5500/
```
